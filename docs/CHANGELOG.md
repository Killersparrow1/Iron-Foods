# Changelog

All notable changes to this project will be documented in this file.

## [0.2.0] - 2026-06-06
### Added
- Complete datasets for all 28 Indian States and 8 Union Territories.
- Validated common food categories (fruits, vegetables, beverages, etc.).
- Enhanced documentation suite.

## [0.1.0] - 2026-06-06
### Added
- Initial project structure.
- Canonical JSON schemas for food, manifest, and search index.
- Search engine implementation.
- Basic regional datasets (Tamil Nadu, Kerala, Karnataka).
