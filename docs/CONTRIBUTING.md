# Contributing to Iron Foods

Thank you for your interest in contributing to the Iron Foods database! We welcome contributions from developers, nutritionists, researchers, and anyone passionate about making Indian food data accessible.

## How to Contribute
1. **Fork the repository.**
2. **Create a new branch** for your feature or data addition.
3. **Follow the [Dataset Guidelines](DATASET_GUIDELINES.md).**
4. **Run validation tests** (`npm run validate`).
5. **Submit a pull request** with a clear description of your changes.

## Types of Contributions
- Adding new food entries to regional or common datasets.
- Improving existing nutrition data with better source attribution.
- Adding translations for food names.
- Enhancing the search engine or indexing scripts.
- Improving documentation.

See [CONTRIBUTORS.md](CONTRIBUTORS.md) for contributor recognition details.
