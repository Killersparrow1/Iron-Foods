# Review Process

All contributions to Iron Foods are subject to review.

1. **Submission:** Contributor submits a Pull Request (PR).
2. **Automated Validation:** GitHub Actions automatically validate the PR against the schema and check for duplicates.
3. **Manual Review:** A maintainer reviews the submission for:
    - Data accuracy
    - Source traceability
    - Multilingual and alias correctness
4. **Action:**
    - **Approve:** PR is merged.
    - **Request Changes:** Feedback is provided to the contributor.
    - **Decline:** The PR is closed if it does not meet quality requirements or is not actionable.
