# Source Policy

We prioritize transparency and traceability for all nutrition data in the Iron Foods database.

## Source Tiering
Data is prioritized based on the reliability of the source:

- **Tier 1 (Institutional):** IFCT (Indian Food Composition Tables)
- **Tier 2 (Government/International):** USDA FoodData Central, FSSAI publications
- **Tier 3 (Academic):** Peer-reviewed research papers
- **Tier 4 (Community/Derived):** Verified community-submitted data (requires mandatory maintainer review)

## Requirements
- All submissions must provide a traceable source reference (e.g., Table 7.2, database name, URL).
- Submissions without verifiable sources will be rejected.
