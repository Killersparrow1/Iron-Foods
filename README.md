Iron Foods is a community-driven, open-source nutrition database dedicated to providing high-quality, structured nutritional information for Indian cuisine.

## 🚀 Getting Started
Iron Foods is designed for developers, researchers, and fitness applications. 
- **[Contribute](docs/CONTRIBUTING.md):** Join our community and help improve the database.
- **[Roadmap](docs/ROADMAP.md):** See our project goals.
- **[Changelog](docs/CHANGELOG.md):** Stay updated with the latest data and features.

---

## 📜 Policies & Guidelines

### Community Guidelines
Iron Foods strives to be a helpful and collaborative community.
- **Expectations:** Be respectful and supportive of all community members. Focus on constructive feedback regarding data quality. Adhere to the [Code of Conduct](docs/CODE_OF_CONDUCT.md).
- **Communication Channels:** Use GitHub Issues for bugs/feature requests, and GitHub Discussions for broader project topics.

### Security
If you discover a security vulnerability in Iron Foods, please report it privately via email to the maintainers rather than opening a public issue.
- **Security Practices:**
    - All data submissions are validated against a strict schema.
    - We do not store any user-sensitive information.
    - All contributions are reviewed by maintainers to prevent malicious data injection.

### License
Iron Foods data is licensed under the [Creative Commons Attribution 4.0 International (CC BY 4.0)](https://creativecommons.org/licenses/by/4.0/) license.
- You are free to share and adapt the data, provided you give appropriate credit.

### Disclaimer
The Iron Foods database is provided for **educational and informational purposes only**.
- **Not Medical Advice:** The nutrition values provided are estimates and not professional medical/dietary advice.
- **Data Variability:** Values can vary based on preparation and sourcing.
- **Public Sources:** Data is derived from public sources (e.g., IFCT-2017, USDA).
- **Consult Professionals:** Always consult a registered dietitian for health decisions.
